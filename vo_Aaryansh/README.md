## How to compile?
Provided with this folder is a CMakeLists.txt file, which you can use to directly compile the code as follows:
```bash
mkdir build
cd build
cmake ..
make
```

## How to run? 
After compilation, type the following:

#### Optical Flow
```bash
./OpticalFlow
```

#### Optical Flow(Dense)
```bash
./OpticalFlow_Dense
```

#### Visual Odometry
```bash
./Visual_Odometry
```
